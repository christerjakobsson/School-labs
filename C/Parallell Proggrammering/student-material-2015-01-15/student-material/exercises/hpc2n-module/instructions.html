Exercise: The module system on Abisko

Prerequisites

Before you start, you should do the following:

  1. Learn how to sign in to Abisko.
  2. Refresh your knowledge of environment variables in Linux.

Objectives

After completing this exercise you should be able to

  1. Load and unload modules on Abisko.
  2. Find available modules and information on how to use them.

Instructions

  1. Head over the Resources section and go to the HPC2N support page on modules.
  2. Read the page carefully and ask a teacher or class mate if you have any questions.

Test

Test your understanding by answering the following questions:

  • How to load the psc module?
  • How to unload the psc module?
  • How to get help on the openmpi/gcc module?
  • How to list the modules that you could load?
  • How to list the modules that you have already loaded?
