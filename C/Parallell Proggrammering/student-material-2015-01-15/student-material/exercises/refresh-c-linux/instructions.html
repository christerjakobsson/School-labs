Exercise: Refresher in C and Linux

Prerequisites

Before you start, you should do the following:

  1. Check that you have registered on the course.

Objectives

After completing this exercise you should once again be able to

  1. Read and write C programs.
  2. Use common Linux commands.

Instructions

Refresh your skills in C programming

This course assumes that you have an intermediate knowledge of the C programming language, specifically the C99 dialect.

For example, you should be able to understand codes like the ones below. 

static void (*func)(void*, int) = NULL;

ptr->next = *(base + off % num);

Head over to the Resources section below and download the PDF "A C Refresher" or open your C programming book and refresh your knowledge and skills in C programming before the course starts.

Refresh your knowledge of Linux commands

You will read and write code written for Linux and run them on fast and powerful parallel machines that run Linux. The only way to access these machines is through a remote shell. It is therefore necessary that you are comfortable working with Linux commands.

Head over to the Resources section below and study at least the introduction and the first five tutorials that you find by following the link "UNIX Tutorial for Beginners".
