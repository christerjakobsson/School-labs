Lecture 5: OpenMP programming

Objectives

After actively participating in this lecture you should be able to

  1. Describe the scope of and some use cases for OpenMP.
  2. Describe the main components of OpenMP: compiler directives and run-time library.
  3. Define the OpenMP concepts of directive and clause.
  4. Define and compare the OpenMP concepts construct and region.
  5. Define the OpenMP concepts of private and shared variable.
  6. Locate the reference documentation for OpenMP.
  7. Write simple OpenMP programs using for loop parallelism and task parallelism.
