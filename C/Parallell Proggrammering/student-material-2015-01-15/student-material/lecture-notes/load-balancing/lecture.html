Lecture 10: Load balancing and termination detection

Objectives

After actively participating in this lecture you should be able to

  1. Define the concept of load balancing and relate it to scheduling.
  2. State and explain a measure of load balance.
  3. Measure the load imbalance in a parallel program.
  4. Explain three types of load balancing schemes: static, dynamic, and adaptive.
  5. Define the termination detection problem.
  6. Explain the need for termination detection algorithms.
  7. State and apply Dijkstra's termination detection algorithm.
