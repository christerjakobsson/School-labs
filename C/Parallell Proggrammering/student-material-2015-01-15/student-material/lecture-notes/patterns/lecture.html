Lecture 12: Parallel programming patterns

Objectives

After actively participating in this lecture you should be able to

  1. Describe the following parallel programming patterns and give examples of their use: perfectly parallel, fork-join, work pool, master-slave, divide and conquer, and pipeline.
  2. Evaluation the choice of a suitable parallel programming pattern for a given application.
  3. Apply the parallel programming patterns to parallel programs.
  4. Analyze an implemented parallel program to identify its parallel programming pattern.
