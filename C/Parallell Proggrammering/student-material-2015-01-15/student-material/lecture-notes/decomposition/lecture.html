Lecture 4: Task decomposition and data distribution

Objectives

After actively participating in this lecture you should be able to

  1. Define the data distribution concept.
  2. Describe the block, cyclic, and block-cyclic array distributions and their multi-dimensional generalizations.
  3. Write a small MPI program that uses array distributions.
  4. Define the task decomposition concept.
  5. Describe the following task decomposition techniques: recursive decomposition, data decomposition, task decomposition, exploratory decomposition, and speculative decomposition.
  6. Explain the concept of task granularity and its impact on a parallel program's performance.
  7. Write small programs based on data decomposition.
