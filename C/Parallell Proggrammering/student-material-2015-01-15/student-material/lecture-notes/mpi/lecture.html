Lecture 8: MPI programming

Objectives

After actively participating in this lecture you should be able to

  1. Describe the distributed memory programming paradigm.
  2. Describe the scope of and some use cases for MPI.
  3. Describe the main components of MPI: library API, compiler wrapper, and launcher.
  4. Define the MPI concepts of communicator, rank, tag, message, message envelope, collective communication, and derived datatypes.
  5. Describe the message-passing semantics in MPI.
  6. Locate the reference documentation for the MPI library.
  7. Write simple MPI-based programs using point-to-point and collective communication operations.
  8. Describe how to run MPI-based programs on the Abisko cluster.
