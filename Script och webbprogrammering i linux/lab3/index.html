Install Plex Media Server 0.9.11 Playback System On Ubuntu 14.04

Filed under  Announcements, Download, FAQs, How-To, News, Open source, Releases, Ubuntu
0

Getting to know about in this tutorial how to install latest release stable version of Plex Media Server 0.9.11 media playback system Via PPA in Ubuntu, Linux Mint, Pinguy OS, Elementary OS, LXLE, Linux Lite and Peppermint and other derivative systems.

The way in installation guide, first we need to know, what is the Plex Media Server and latest updates for? It is a centralized home media playback system with a powerful central server–the Plex Media Server–that streams its media to many Plex player Apps. The Server is available on many platforms like Windows, OS X, and many flavors of Linux, as well as many NAS devices like ReadyNAS or Synology. You use an App to view your media library and watch shows. Apps are available for PCs, mobile devices like smartphones and tablets, and many Smart TVs.

All eyes on the latest released version of Plex Media Server is 0.9.11, now its available for Ubuntu 14.04, Ubuntu 14.10 and other derivatives systems to install, update and download also. This new version coming with new changes and several updates.

  • Rapid seeking could lead to video playback stopping.
  • Improved reliabilty around publishing network connections.
  • Use TLS instead of SSLv3 when talking to plex.tv.
  • (DASH) More reliable seeking.
  • (ReadyNAS) Fix logo not appearing.

How to Install Latest Plex Media Server 0.9.11 in 32-bit Ubuntu 14.04, Ubuntu 14.10 Linux Systems: Step-by-Step Procedure

sudo apt-get install gdebi

wget https://downloads.plex.tv/plex-media-server/0.9.11.1.678-c48ffd2/plexmediaserver_0.9.11.1.678-c48ffd2_i386.deb

sudo gdebi plexmediaserver_0.9.11.1.678-c48ffd2_i386.deb

How to Install Latest Plex Media Server 0.9.11 in 64-bit Ubuntu 14.04, Ubuntu 14.10 Linux Systems: Step-by-Step Procedure

sudo apt-get install gdebi

wget https://downloads.plex.tv/plex-media-server/0.9.11.1.678-c48ffd2/plexmediaserver_0.9.11.1.678-c48ffd2_amd64.deb

sudo gdebi plexmediaserver_0.9.11.1.678-c48ffd2_amd64.deb
